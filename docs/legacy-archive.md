# Legacy Forum Archive

This section contains a collection of discussion threads from the old Holy City Audio phpBB forum, saved as PDFs. Some threads originally included file attachments; those attachments may be in the `patches/` folder one level up in the source repository.

In early 2016 the forum site sustained an external hack, which required extensive recovery measures. Unfortunately one casualty of that recovery was that all attachments wound up getting corrupted, so attachments prior to 2016 may be missing or unreadable.

Some of this content may be obsolete or refer to features that have since changed.
