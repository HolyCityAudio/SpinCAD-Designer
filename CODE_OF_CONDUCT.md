# Code of Conduct

## Our Pledge

SpinCAD Designer is a small, independently maintained open source project. We are committed to providing a welcoming and respectful environment for everyone who participates — whether by filing issues, submitting pull requests, asking questions, or engaging in discussion.

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. By participating, you agree to uphold its standards.

## Expected Behavior

- Be respectful and constructive in all interactions
- Assume good intent — most misunderstandings are just that
- Stay on topic in issues and pull requests
- Accept that response times may be slow; this is a one-person project maintained on a volunteer basis

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

This is a small project with a single maintainer. There is no formal review board or dedicated enforcement team. In practice, enforcement uses the tools GitHub provides:

- **Comment moderation:** Off-topic, abusive, or disruptive comments may be edited, hidden, or deleted
- **Thread locking:** Unproductive discussions may be locked
- **User blocking:** Participants who repeatedly violate this code of conduct may be blocked from the repository
- **GitHub reporting:** Serious or platform-level abuse can be reported directly to [GitHub Support](https://support.github.com/contact/report-abuse)

If you experience or witness behavior that violates this code of conduct, you can flag it by opening a GitHub issue labeled `conduct`, or by using GitHub's built-in comment reporting tools.
